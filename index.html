Date Object

Once an instance of the object you want to refer to is created, you can use the date object to set specific time values, as well as access certain time values for use in scripts.

Once the button is pressed below, an instance of the date object is created. The methods getDate(), getMonth(), and getFullYear() are then applied to the new instance to create variables that set the date in a DD/MM/YYYY format. The methods getMinutes() and getHours() are applied to the instance to create new variables that set the time in a HH:MM AM/PM format.
